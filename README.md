# Sales & Demand Forecasting for Businesses

This project was developed as part of the Future Interns Machine Learning Internship.

Objective:
Forecast future sales using historical business data.

Tools Used:
- Python
- Pandas
- Matplotlib
- Scikit-learn

Features:
- Data preprocessing
- Feature engineering
- Sales forecasting
- Performance evaluation
- Business insights
- Data visualization

Evaluation Metrics:

- MAE: 270.74
- RMSE: 811.69
- R² Score: 0.01

Business Insights:

- Sales vary across months.
- Certain regions generate more revenue.
- Product categories influence overall sales.
- Sales forecasting can help inventory planning.
- Businesses can use these insights to optimize operations.
